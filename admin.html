🏪 Setup Kasir Warung

Panel administrasi untuk setup awal aplikasi kasir warung dengan Google Apps Script backend

📋 Langkah-Langkah Setup

1 Buat Google Spreadsheet

Buat spreadsheet baru di Google Drive dengan nama Kasir Warung Database

2 Copy Spreadsheet ID

Salin ID spreadsheet dari URL:
https://docs.google.com/spreadsheets/d/SPREADSHEET_ID_DISINI/edit

3 Setup Google Apps Script

Buka spreadsheet → Extensions → Apps Script → Paste kode kode.gs → Ganti SPREADSHEET_ID dengan ID Anda

4 Initialize Spreadsheet

Jalankan fungsi initializeSpreadsheet() di Apps Script untuk membuat struktur sheet

5 Deploy Apps Script

Deploy → New deployment → Web app → Execute as: Me → Who has access: Anyone → Copy deployment URL

6 Update Frontend

Edit app.js → Ganti BACKEND_URL dengan deployment URL → Set useBackend = true

🔧 Initialize Database

Klik tombol di bawah untuk menjalankan setup awal spreadsheet (buat sheet Users, Menu, Transaksi, Settings)
